Project :-
----------
This code provides the solution to the Predict Office Space Price challenge on www.Hackerrank.com

Link :- (https://www.hackerrank.com/challenges/predicting-office-space-price)

It uses normal equations after mapping the features to higher order polynomials to hypothesis multivariate 
model parameters to predict an estimate of the office space prices given a set of features to train the model.
